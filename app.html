what's nazon

nazon is simple javascript framework

nazon module

Define Controller
nazon.actionController('name_space', function(){/* write biz */})
Define Module
nazon.addModule('module_name', {/* define module component */})
Load Module
var moduleA = nazon.require('module_name');

nazon api

Effect API
show : nazon.get('li a').show()
hide : nazon.get('li a').hide()
Debug API
debug : nazon.get('li a').dump()
Selector API
CSS1 is full Support : nazon.get('li')
CSS2 is full Support : nazon.get('li a')
CSS3 is full Support : nazon.get('li:not(#demo4)')
CSS4 is only (:matches, not(expr, expr..)) Support : nazon.get('ul :matches(#demo4, #demo5) a')
